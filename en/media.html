For media and institutions

This page provides a clear public briefing for editorial teams, organizations and institutions. The wording remains legally cautious and anonymization safeguards are maintained.

Publicly available

  • anonymized summaries and timeline,
  • public evidence index references,
  • source links to Markdown documentation.

Media and institutional contact path

There is currently no dedicated public media email published in this repository.

After justified contact, fuller supporting materials may be shared with verified media or institutional reviewers using a secure process.

Role of GitHub Issues/Discussions

  • GitHub Issues/Discussions are for technical corrections and repository feedback only.
  • Older test Issues are not media contact channels.

Legal and privacy limits

  • no publication of sensitive personal data,
  • material-sharing scope depends on purpose and data-protection compliance,
  • the report presents allegations/claims with documentation and is not a final legal determination.
